Co je Vorb?


Vorb je vanilla server, jehož cílem je poskytnout autentický herní zážitek bez pluginů, které by narušovaly zábavu ze hry.


Proč vznikl?


Tento server jsme vytvořili, protože většina ostatních serverů na československé scéně je příliš modifikovaná. Chtěli jsme jednoduchý a čistý zážitek ze hry, přesně takový, jaký zamýšleli vývojáři.


Jak se připojit?


Na server se dá připojit přímo v herním server listu. Stačí vyhledat "Vorb" nebo ho najít mezi ostatními československými servery.
